# Memloop

> 簡單的單字卡系統，根據你的記憶狀態自動安排複習，讓你用更少的時間記住更多內容。

## 🎯 專案介紹

Memloop 是一個類似 Anki 的間隔重複（Spaced Repetition）網頁應用，專為有高效語言學習或記憶需求的使用者設計。不需要安裝任何 App，直接在網頁上就能記錄並學習單字。

系統會根據你對每一張卡片的熟悉程度，動態調整下一次出現的時機：
- **不熟（困難）** → 隔天馬上複習
- **普通（正常）** → 3 天後複習
- **熟悉（簡單）** → 5 天後複習

這將大幅減少你對已熟悉單字的無效重複時間，將注意力專注在難以記住的學習內容上。

## 🚀 核心功能

- **自動排程演算法**：根據「簡單」、「正常」、「困難」調整複習頻率。
- **每日限額**：每天推播最高 30 張單字挑戰，持之以恆無壓力。
- **Leech (難記卡) 標示**：錯誤達 8 次的卡片會被標記。
- **純前端運作**：所有進度與卡片資料均儲存在瀏覽器的 `localStorage` 中。不會有隱私外洩風險。
- **Markdown 擴充字庫**：一鍵匯入 `.md` 單字表，輕鬆累積專屬您的單字本。

## 🛠️ 開發技術

- HTML5
- CSS3 (現代 Glassmorphism 玻璃擬真介面、Mobile First 響應式排版、3D 卡片翻轉)
- Vanilla JavaScript (ES6+)

## 📝 Markdown 匯入格式範例

您可以透過上傳 `.md` 檔案擴充您的單字庫。檔案中的表格必須符合以下格式（表頭必須有且包含特定分隔線）：

```markdown
| 漢字 / 假名 | 詞性 | 中文意思 | 備註 |
| :--- | :--- | :--- | :--- |
| 出会う（であう） | 第一類動詞 | 相遇、相逢 | |
```
